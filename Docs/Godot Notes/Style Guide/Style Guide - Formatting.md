### Encoding and special characters[¶](https://docs.godotengine.org/en/stable/tutorials/scripting/gdscript/gdscript_styleguide.html#encoding-and-special-characters "Permalink to this headline")

-   Use line feed (**LF**) characters to break lines, not CRLF or CR. _(editor default)
-   Use one line feed character at the end of each file. _(editor default)_
-   Use **UTF-8** encoding without a [byte order mark](https://en.wikipedia.org/wiki/Byte_order_mark). _(editor default)_
-   Use **Tabs** instead of spaces for indentation. _(editor default)_
