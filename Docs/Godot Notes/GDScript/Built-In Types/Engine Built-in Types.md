#gdscript 
### Engine built-in types[¶](https://docs.godotengine.org/en/stable/tutorials/scripting/gdscript/gdscript_basics.html#engine-built-in-types "Permalink to this headline")

#### [Color](https://docs.godotengine.org/en/stable/classes/class_color.html#class-color)[¶](https://docs.godotengine.org/en/stable/tutorials/scripting/gdscript/gdscript_basics.html#color "Permalink to this headline")

Color data type contains `r`, `g`, `b`, and `a` fields. It can also be accessed as `h`, `s`, and `v` for hue/saturation/value.

#### [NodePath](https://docs.godotengine.org/en/stable/classes/class_nodepath.html#class-nodepath)[¶](https://docs.godotengine.org/en/stable/tutorials/scripting/gdscript/gdscript_basics.html#nodepath "Permalink to this headline")

Compiled path to a node used mainly in the scene system. It can be easily assigned to, and from, a String.

#### [RID](https://docs.godotengine.org/en/stable/classes/class_rid.html#class-rid)[¶](https://docs.godotengine.org/en/stable/tutorials/scripting/gdscript/gdscript_basics.html#rid "Permalink to this headline")

Resource ID (RID). Servers use generic RIDs to reference opaque data.

#### [Object](https://docs.godotengine.org/en/stable/classes/class_object.html#class-object)[¶](https://docs.godotengine.org/en/stable/tutorials/scripting/gdscript/gdscript_basics.html#object "Permalink to this headline")

Base class for anything that is not a built-in type.
